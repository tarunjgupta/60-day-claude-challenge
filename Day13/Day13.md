# 🚀 Day 13 – Claude Connectors & AI Job Discovery

## 📌 Overview

On Day 13 of the **#60DayClaudeChallenge**, I explored how **Claude Connectors** can be used to automate job discovery and perform intelligent career analysis.

Instead of manually searching for jobs, I connected my Indeed profile with Claude and used AI to:

* Discover relevant job opportunities
* Analyze market demand
* Identify skill gaps
* Get personalized career recommendations

---

## 🔗 What are Claude Connectors?

Claude Connectors allow AI to connect with external platforms like job portals and use that data for deeper, personalized insights.

### Key Benefits:

* 🔗 Connect external platforms directly to Claude
* 🤖 Automate job discovery
* 📊 Analyze hiring trends and in-demand skills
* 🎯 Improve career planning with data-driven insights

---

## 🧠 Prompts Used

### 📍 Prompt 1: Professional Profile

```
PROMPT 1: Professional Profile

Describe your professional background, including:

Current role
Years of experience
Key skills and technologies
Industry/domain expertise
Current company type
Current location
Notable achievements, certifications, or accomplishments
```

### 📍 Prompt 2: Job Search Criteria

```

PROMPT 2: Job Search Criteria

Specify your target job requirements, including:

Desired job titles
Preferred company types
Preferred locations (Remote/Hybrid/Onsite)
Salary expectations
Industries or companies to exclude
Job posting recency requirements
Any additional preferences or constraints

```

### 📍 Prompt 3: Job Discovery & Analysis

```

PROMPT 3: Job Discovery & Analysis

Using my professional profile and job search criteria:

Search for matching job opportunities using the available job connector(s)
Prioritize the highest-fit roles
Exclude jobs that do not meet my requirements
Return the top opportunities in a table containing:

Company
Role
Location
Posted Date
Direct Application Link
Match Score
Why It Fits My Profile
CTC

Also provide:

Most commonly required skills across the jobs
Skill gap analysis
Market demand insights
Recommendations to improve my chances of getting interviews
Overall fit assessment for my target roles and compensation goals.
```

---

## ⚙️ Step-by-Step Guide (Indeed + Claude Connection)

### Step 1: Open Claude

* Go to Claude AI platform

### Step 2: Set Effort Level

* Set effort level to **Low** (for faster responses)

### Step 3: Open Connectors

* Navigate to **Claude Connectors**

### Step 4: Connect Indeed

* Use the provided connector link given below
* Authorize access to your Indeed account

### Step 5: Start New Chat

* Open a fresh conversation in Claude

### Step 6: Create Profile

* Paste **Prompt 1**
* Generate your professional profile

### Step 7: Define Job Criteria

* Paste **Prompt 2**
* Specify roles, location, salary, etc.

### Step 8: Discover Jobs

* Paste **Prompt 3**
* Let Claude fetch and analyze jobs using Indeed data

### Step 9: Analyze Results

* Review:

  * Top job opportunities
  * Match scores
  * Salary insights
  * Skill gaps

### Step 10: Save Insights

* Take screenshots
* Save analysis for future reference
