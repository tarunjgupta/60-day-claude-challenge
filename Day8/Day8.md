## 🌍 Project Built

### Personal Environmental Health Analyzer

Today, I built a complete AI-powered environmental dashboard using Claude.

This dashboard analyzes:

* 🌫 Air Quality (AQI, PM2.5, PM10)
* 💧 Water Quality (TDS, Hardness, pH)
* 🧠 Health Impact (lungs, sleep, energy, skin, hair)
* 📊 City comparisons & rankings
* 📈 Interactive charts & filters
* 🎯 Personalized recommendations

It also generates a **Health Score (0–100)** and provides actionable insights.

---

## 🧪 Features of the Dashboard

* 📊 AQI Comparison Charts
* 📈 PM2.5 & PM10 Visualizations
* 🏙 City Ranking System
* 🎛 Interactive Filters (AQI range, city selection)
* 📋 Environmental Health Report Card
* 🚦 Risk Indicators (Low, Moderate, High)
* 💡 Personalized Recommendations
* 📥 Fully Downloadable HTML Application

## 🧠 Key Learnings

### 1️⃣ Claude Artifacts = Real Applications

Claude can generate fully interactive tools, not just text responses.

---

### 2️⃣ Prompt = Product

The way you structure your prompt directly impacts the quality of the output.

Instead of asking simple questions, I gave:

* Role instructions (Analyst + Researcher + Designer + Developer)
* Clear requirements
* Structured output expectations

---

### 3️⃣ AI as a Full-Stack Builder

Claude handled:

* Data Analysis
* UI Design
* Logic Building
* Chart Generation
* HTML Application Creation

---

### 4️⃣ Context Engineering is Powerful

Providing detailed instructions helped in:

* Better design
* Better insights
* More structured output

---

### 5️⃣ From User to Builder Mindset

Instead of consuming AI outputs, I:

* 👉 Built a real-world tool
* 👉 Created something usable
* 👉 Learned how to think like a product builder
